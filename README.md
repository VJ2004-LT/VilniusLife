# Vilnius Interactive Map
Uploaded student project by me (Valentinas Jokšas) and Dmitrij Pikta. 

Done during 2026-02-12 - 2026-06-07.

I decided to make additional improvements to this project after the semester ended because I really liked the idea of the project.

## About
 
This web app aggregates data from Vilnius open APIs into a single interactive map, making city data visual and accessible. It also includes a built-in neighborhood forum where users can make posts and comments about specific areas, and like or dislike them to share their opinion.

## Features
 
- **City Reports** — Visualizes reports from [tvarkaumiesta.lt](https://tvarkaumiesta.lt)
- **Noise Levels** — Real-time noise data from Vilnius open APIs
- **Safety & Traffic** — Traffic-based safety level visualization
- **Priority School Finder** — Finds and visualizes routes to priority schools
- **Neighborhood Forum** — Community posts, comments, and reactions tied to map areas
